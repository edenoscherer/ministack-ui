# MiniStack UI

**A melhor experiência open-source para debugging local de sistemas distribuídos AWS-compatible.**

## O que é?

MiniStack UI é uma plataforma self-hosted focada em:

- **Observabilidade local** — veja o que está acontecendo nos seus serviços em tempo real
- **Debugging distribuído** — rastreie eventos através de múltiplos serviços
- **Runtime inspection** — inspecione filas, tópicos, secrets e logs
- **Developer experience** — interface workflow-first, sem o ruído do AWS Console

## Compatibilidade

- [MiniStack](https://github.com/ministack) — ambiente local AWS-compatible
- [LocalStack](https://localstack.cloud) — emulador AWS
- AWS real (em breve)

## Quick Start

```bash
pnpm install
pnpm dev
```

Acesse `http://localhost:3000` para abrir a interface.
